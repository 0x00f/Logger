Detailed Description

Generates filename strings for each new measurement file.

Uses the system module to get user filename data from NVSRAM

Function Documentation

char* Gen_Filename ( )

Generates a new filename for a new measurement. It either retrieves the filename from backup sram, or if sram is not initialized, uses the default "measurement". The filename has the format: "filename"_n, where n is an integer that is incremented for each new measurement.

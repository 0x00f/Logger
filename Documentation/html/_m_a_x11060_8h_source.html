108 /*
109  * Sampling Instant Control Register (32*n bits)
110  * By default, the devices sample all 4 input channel simultaneously.
111  * To delay the sampling instant on one or more channels, program the appropri-
112  * ate byte in the Sampling Instant Control Register.
113  * The delay of the actual sampling instant of each individ. channel from the
114  * default sampling instant (PHI_[7:0]=0x00) is adjustable between 32 to
115  * 819,121 XIN clock cycles ( 1,3µs to 333µs with Fxinclk at 24,576MHz).
116  */

118 /*
119  * Configuration Register (8 Bits)
120  * The config. register contains 5 bits that control the functionality of the
121  * device. Default state is 0x00.
122  * Bit 7 - SHDN - Set high to shutdown, low for regular operation
123  * Bit 6 - RST - Set high to reset all registers to default, and realign
124  * sampling clocks and output data
125  * Bit 5 - EN24BIT - Set to 0 for MAX11060
126  * Bit 4 - XTALEN - Int osc. enable bit. To use an external clock, set to 0
127  * Bit 3 - FAULTDIS - Overvoltage fault protection disable bit. Set high to
128  * disable overvoltage fault protection.
129  * Bit 2 - PDBUF - Set high to disable internal reference buffer (when an
130  * external ref is used. Set to 0 to enable internal ref.
131  * Bit [1:0] - Reserved. MUST BE SET TO 0.
132  */
